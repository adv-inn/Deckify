Spotify App Setup

Each user needs their own Spotify App (takes ~2 minutes).

  1. Go to developer.spotify.com and create a new app with:
    App name: Deckify   Redirect URI: https://steamdeck.local:39281/callback   Check Web API
  2. Copy the Client ID and enter it in the plugin's Spotify Login flow
  3. When scanning the QR code, your browser may show a certificate warning ‚Äî this is expected (self-signed cert). Proceed to continue.
  4. If you changed your Steam Deck's hostname, replace steamdeck in the redirect URI accordingly
Note: Spotify development mode limits each Client ID to 5 authorized users. You can add users in your app's dashboard.
